Hello World

  • HelloWorld project - is the simple starter-example how to use ClassHidra library - lightweight MVC platform for J2EE development.

  • It's really lightweight - the base package classhidra-base-*.jar = 840k and despite that included: action/model annotations mapping, rest support, async execution, the rich tag library for jsp design, scheduling package, json serializing/deserializing, ...

  • With EJB and CDI plugins implementation is possible to include stateless and stateful EJB objects as model component.

  • From Source Forge is possible to download the different examples (as *.war or *.pom files) of variety of implementation: base, ejb, cdi, jetty,...

Project starter

  • For avoid web.xml into project is included the package starter/ which contains filter+servlet+initializer sufficient for processing

  • StarterController.java
  • StarterFilter.java
  • StarterInitializer.java

  • StarterInitializer contains the initial application parameters.

  • StarterFilter is an extension of the basic ClassHidra filter which intercepts the actions/calls request.

  • StarterController is an extension of the basic ClassHidra controller for processing the incoming request.

Project controllers

  • For avoid web.xml is included the package starter/ which contains filter+servlet+initializer sufficient for processing

  • HelloWorld.java
  • LongPooling.java
  • ControllerRest.java
  • Annotations use

  • HelloWord - is controller which manages the current view.

  • LongPooling - is async controller (deferred) placed as an example how to organize long pooling communication with ClassHidra

  • ControllerRest - stateful rest implementation.
